MENA MURIA CEREMONIAL CLOCK
---------------------------

FILES YOU MUST ADD:
- assets/intro.jpg   → your ceremonial clock image
- icons/icon.png     → your golden ABORU heart icon

HOW TO USE:
1. Place all files in the structure shown.
2. Compress the entire folder into a ZIP.
3. Open index.html on any device.
4. For Android: Add to Home Screen to install as an app.

FEATURES:
- Fade-in intro screen
- Synced analog + digital clock
- Photo-realistic ceremonial design
- ABORU icon as app symbol
